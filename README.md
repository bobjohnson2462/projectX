## Проект 6: Система управления библиотекой

### Описание
Система учета книг, читателей, выдачи и возврата литературы с расчетом штрафов.

### Требования к функционалу

#### Разрешено:
- Каталог книг с автором, названием, годом издания, жанром, количеством экземпляров
- Регистрация читателей с ФИО, контактами, датой регистрации
- Выдача книги читателю на фиксированный срок (14 дней по умолчанию)
- Продление срока выдачи при отсутствии резервов на книгу
- Резервирование книги при её отсутствии в наличии
- Расчет штрафа за просрочку (фиксированная сумма за каждый день)
- Поиск книг по автору, названию, жанру
- Отчеты: популярные книги, должники, оборот книг

#### Запрещено:
- Выдача несуществующей или отсутствующей в наличии книги
- Выдача книги читателю с долгом по штрафам свыше порога
- Отрицательный срок выдачи или штраф
- Удаление книги из каталога при наличии активных выдач
- Изменение данных о завершенной транзакции выдачи/возврата

#### Граничные случаи:
- Одновременная попытка выдачи последнего экземпляра книги двумя читателями
- Возврат книги с повреждениями — дополнительный штраф
- Продление срока при появлении резерва во время активной выдачи

| 3 | Механизм выдачи и возврата книг | Неделя 3 |
| 4 | Система резервирования | Неделя 4 |
| 5 | Расчет штрафов и управление долгами | Неделя 5 |
| 6 | Генерация отчетов и статистики | Неделя 6 |
